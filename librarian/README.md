# librarian

Детерминированный конвейер «книга → Markdown-главы» для работы с LLM.
Любой входной формат (FB2, EPUB, DOCX, HTML, TXT/MD, PDF с текстовым слоем) →
каталог чистых глав с подсчитанными токенами. Без сети, без LLM, без случайности:
одинаковый вход даёт побайтово идентичный выход.

## Установка

    pipx install ./librarian-*.whl     # или: uv tool install .

Офлайн-контракт: рантайм и установка из локального wheel не требуют сети
(словарь токенизатора вендорен в пакет).

## Быстрый старт

    lib ingest книга.epub роман.fb2 статья.html
    lib list
    lib list <book-id>
    lib get <book-id> 1-3,7
    lib get <book-id> --budget 12000

## Команды

| Команда | Назначение |
|---------|-----------|
| `lib ingest <файлы…> [--force] [--no-keep-source] [--config cfg.toml] [--verbose]` | Обработать книги в библиотеку; таблица «файл · id · статус · score». |
| `lib list [<book-id>]` | Без аргумента — все книги; с id — главы книги (n, заголовок, токены, summary). |
| `lib get <book-id> <spec>` | Печать глав по диапазону (`1-3,7`) в stdout. |
| `lib get <book-id> --budget N [--from K]` | Жадно подряд главы начиная с K, пока сумма токенов ≤ N. |
| `lib info <book-id>` | JSON: метаданные книги + метрики качества. |
| `lib doctor [<book-id>]` | Без id — книги в review и битые каталоги; с id — отчёт по книге. |
| `lib reingest --all [--config cfg.toml] [--verbose]` | Пересобрать библиотеку из `source/` текущим кодом/конфигом. |
| `lib rm <book-id>` | Удалить книгу и пересобрать индекс. |

Корень библиотеки: `--library <путь>` (или переменная `LIB_HOME`, по умолчанию `./library`).
Данные идут в stdout, диагностика — в stderr. Коды выхода: `0` — успех, `1` — ошибка
выполнения, `2` — ошибка использования.

## Качество

Каждая книга получает score из пяти метрик (coverage, structure, garbage, encoding,
dehyphen): `ok` (score ≥ 0.90 и без жёстких триггеров) — молча в библиотеку; `review`
(0.60 ≤ score < 0.90 либо есть триггеры) — сохранена с предупреждением, детали в
`lib doctor <id>`; `failed` (score < 0.60) — не сохраняется. Сканы и PDF под паролем
честно дают failed (OCR и снятие DRM — вне задач инструмента).

## Конфигурация

`lib ingest --config cfg.toml`; пороги и параметры резки — §14 спеки. Правки метаданных
руками: поставить `"meta_locked": true` в `book.json` — реингест их не перетрёт.
`lib reingest --all` пересобирает библиотеку текущим кодом/конфигом; книги, у которых
ничего не изменилось, пропускаются с сообщением «уже в библиотеке» — выход и так
детерминирован.

## Ограничения

- PDF: типографски обычные книги; сложная вёрстка может уйти в review.
- DRM не обходится; правомерность источников — ответственность пользователя.
- Сетевые ФС (NFS/SMB) не поддерживаются (advisory-lock ненадёжен).
- MOBI/DJVU/сканы-без-текста — нет (v3: OCR, calibre).
